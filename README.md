# Shakib Ahmed – Portfolio

React + Vite single-page portfolio for **Shakib Ahmed** – Software Engineer, AI Engineer, and Entrepreneur, Managing Director at **Bangladesh Software Company Limited (BSCL)**.

## Tech Stack

- React 18
- Vite
- Modern, custom CSS (no UI framework needed)

## Getting Started

From the project root (`D:\sk-portfolio`):

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (by default `http://localhost:5173`).

## Customization

- Profile photo: `images/sk.jpeg` (already wired into the hero section)
- Main content: edit `src/App.jsx` to change text, sections, or add real project links and contact info.
